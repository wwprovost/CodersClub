Here's a big new idea: you can define a procedure as a body of code that you can then call from sonewhere else in your program. And procedures can have parameters, so that the callers can supply different inputs to the procedure's logic. In JavaScript, a procedure is really just a function that doesn't return a value.

In this program the printXs procedure takes a parameter howMany, and will print a line of 'x' characters of whatever length the caller requests.

Once you've defined a procedure, you call it by name, and pass values for the parameters in parentheses.

procedure printXs(howMany) { repeat howMany times { print "x"; } new line; } call printXs(10); call printXs(2); call printXs(32);
def printXs(howMany): line = "" for x in range(howMany): line = line + "X" print line printXs(10) printXs(2) printXs(32)
function printXs(howMany) { let line = ""; for (let x = 0; x < howMany; ++x) { line = line + "X"; } print(line); } printXs(10); printXs(2); printXs(32);
