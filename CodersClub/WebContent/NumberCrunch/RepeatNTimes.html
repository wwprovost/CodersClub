You can make some code run multple times, in a loop. There are a few ways to do this. We'll look first at repeating a fixed number of times.

repeat N times is followed by a block of code -- so, again, you must use curly braces -- and will execute that code the number of times you specify.

for var in range(N): is followed by an indented line or block of lines, and will run that code the number of times you specify.

for (let var = 0; var> < N; ++var) is followed by a block of code -- so, again, you must use curly braces -- and will execute that code the number of times you specify.

This can be a powerful tool for repeating math operations. The middle part of the program below calculates the value of doubling the number 1, 10 times -- also known as two to the power of 10.

Or, sometimes it's useful to know which "iteration" of the loop you're in, and so you create a counter.

You can use the variable itself in the loop, too -- it's known as the index value. Notice that the count starts at zero, not one.

repeat 5 times { print "X "; } new line; number x = 1; repeat 10 times { x = x * 2; } print line "2 to the 10th is " + x; number count = 0; repeat 3 times { count = count + 1; print line "Iteration " + count; }
line = "" for count in range(5): line = line + "X " print line x = 1 for count in range(10): x = x * 2 print "2 to the 10th is", x for count in range(3): print "Iteration", count
