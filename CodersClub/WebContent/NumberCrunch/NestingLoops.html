You can put any statements in a block that you could put in your program generally. That means that you can "nest" conditionals and loops.

This program makes a grid of 5 rows and 5 columns. Try running it. Then, try replacing the "5" in the second loop with the variable count. See how, because count varies each time, the rectangular shape turns into a triangle?

number count = 1; repeat 5 times { repeat 5 times { print "x"; } new line; count = count + 1; }
for row in range(5): line = "" for column in range(5): line = line + "x" print line
for (let count1 = 0; count1 < 5; ++count1) { let line = ""; for (let count2 = 0; count2 < 5; ++count2) { line = line + "x"; } print(line); }
