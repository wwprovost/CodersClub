NCL lets you define number variables, which represent values that you can change during the program. You declare variables by stating its type (a "number"), name, and starting value.

Python lets you define variables, which represent values that you can change during the program. You declare variables by stating the variable name and setting it to a equal starting value.

JavaScript lets you define number variables, which represent values that you can change during the program. You declare variables with the keyword let, followed by the variable name and starting value.

Then, you can change the value of a variable over time -- just by giving it a new value, or by using arithmetic. Variable names are "case-sensitive," which means that however you write the variable name initially, you have to write it that way after that. So you can't call it "count" and then later call it "COUNT" or "Count" (or "cOuNt" -- but why would you do that?).

You can also print a few things on one line, just by separating them with commas:

number count = 1; print line "Count is " + count; count = 2; print line "Count is " + count;
count = 1 print "Count is", count count = 2 print "Count is", count count = 3 print "Now, there are", count, "things."
let count = 1; print("Count is " + count); count = 2; print("Count is " + count);
