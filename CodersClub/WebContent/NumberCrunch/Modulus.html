Now we'll add an operator to the language: the % operator, also called the modulus. For positive numbers, this gives the remainder resulting from dividing one number by the other -- but just the remainder, where we don't care about the quotient, or get the quotient elsewhere.

So 13-modulus-5 is 3, 11-modulus-2 is 1. And now, there is an easier way to tell if one number is a factor of another, because if it is, then the bigger number modulus the smaller one will be zero.
