Now we add a new kind of loop to the language, known as a for loop. This lets you define a variable, make that variable start at one value, and then either climb or descend to another value. For the starting value, each value in-between, and the end value, the loop block will be executed.

It will be useful for upcoming challenges to be able to run a loop with an index from one value to another. Python makes this easy, as you can define the range in your loop with a starting and ending value. The only strange bit is, the loop will execute from the starting value to one less than the ending value.

It will be useful for upcoming challenges to be able to run a loop with an index from one value to another. You can define both the starting and ending values in your for loop. You also control, using either the <= or the <, whether the ending value is included in the loop, or not.

for x = 1 to 5 { print "" + x + " "; } new line;
line = "" for x in range(1,5): print x
let line = ""; for(let x = 1; x <= 5; ++x) { line = line + x + " "; } print(line); print("");
