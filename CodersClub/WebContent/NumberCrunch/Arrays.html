Another big, new idea: an array or list can hold multiple values, each of which has a numerical index. Then you can work with all of the values, one at a time in a loop.

The example below creates an array of 7 numbers, and then sets each one of them to the high temperature for each day in a week. Then it loops over the data, and calculates the average temperature.

Note that, while NCL has arrays, they are clumsy to use. This might be a good time to switch over to using Python or JavaScript: as you'll see, they make arrays a lot easier.

number[7] temps; temps[1] = 50; temps[2] = 52; temps[3] = 61; temps[4] = 65; temps[5] = 55; temps[6] = 54; temps[7] = 61; number sum = 0; for day = 1 to 7 { sum = sum + temps[day]; } number average = sum / 7; print line "The average high temperature this week was " + average;
temps = [ 50, 52, 61, 65, 55, 54, 61] sum = 0 for temp in temps: sum = sum + temp average = sum / 7 print "The average high temperature this week was", average;
let temps = [ 50, 52, 61, 65, 55, 54, 61]; let sum = 0; for (let temp of temps) { sum = sum + temp; } let average = sum / 7; print("The average high temperature this week was " + average);
