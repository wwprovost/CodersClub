You can make a program do different things based on inputs or random results, with a conditional structure, using the keywords if and else. Use if and a logical test to make a bit of code run only if the test is passed. Logical tests are things like "does A equal 5?" or "is X greater than Y?" Notice that we use == to check if one number equals another -- not the single =. And use != for "is not equal to."

After an if and the test, you must use curly braces { and } around the code you want to run conditionally. The code inside the braces is known as a "block" of code. You can put things all on one line, or on separate lines -- whatever seems right to you.

After an if and the test, you must include a colon :, and the "block" of code you want to run conditionally must go on the following line or lines. That line or all of those lines must be indented -- must start with some spaces or tabs -- in order to be controlled by the if statement.

After an if, test, and the block of code, you may write an else and a different block, and this second block will be run only if the test turns out not to be true -- for example when A is 10, meaning that it is not greater than 10.

number A = 1; A = A + 4; if A == 5 { print line "A equals 5."; } A = A * 2; if A > 10 { print line "A is greater than 10."; } else { print line "A is either 10, or less than 10."; }
A = 1 A = A + 4 if A == 5: print "A equals 5." A = A * 2 if A > 10: print "A is greater than 10." else: print "A is either 10, or less than 10."
let A = 1; A = A + 4; if (A == 5) { print("A equals 5."); } A = A * 2; if (A > 10) { print("A is greater than 10."); } else { print("A is either 10, or less than 10."); }
