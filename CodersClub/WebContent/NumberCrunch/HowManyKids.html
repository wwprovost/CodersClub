Write a program that solves the following problems. First, if there are 20 kids in a class, and 4 classes in each grade, and 6 grades in the school ... then how many kids attend the school?

Then, for a field trip, we can get 60 kids on a school bus. How many buses will we need?

Finally, we'll need one chaperone for each class, plus one for each bus. How many chaperones do we need?
