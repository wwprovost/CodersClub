Write a program that finds the first 20 prime numbers. A prime is a positive whole number, or "counting number," for which there are no numbers other than one and the number itself that you can multiply to get the number. So, 2 is prime; 3 is prime; 4 is not prime because 2 * 2 = 4; 5 is prime, 6 is not (and all even numbers over 2 are not); 7 is prime, etc.

Finding primes requires nested loops. First, your outer loop sets the number to be tested, starting from 1, or 2, something like that. Then, to test, you have to run from 2 up to, say, half of the number itself, and then see if the one number is a multiple of the other. If the loop "falls through," then the number must be prime, so you print it and count it as one of your 20.
