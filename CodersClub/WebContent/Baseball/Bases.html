Now write a function bases_occupied. Just like runs_scored(), this will take a hits and a bases parameter. But this function will modify bases and return the updated list.

You'll need to loop through the list of bases, and for each occupied base (value is True), "move" that runner ahead. A few tricks to this:

  • Some runners will move off the end of the list: those are the ones who cross home plate and score runs, and so we don't track them in this list anymore. In other words, they're no longer on base.
  • You'll have an easier time if you work backwards from 3rd base to 1st. If you loop forwards, you might move a runner onto a base already occupied by another runner -- or move the same runner forward, over and over.

Remember to put the batter who got the hit on base, too.

Un-comment the starter code to test your function.
