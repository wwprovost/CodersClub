Now, develop a function at_bat() that can decide what happens on a single at-bat. Generate a random number and return 0, meaning the batter is out, 2/3 of the time. For the other 1/3 of the time, generate a second random number and return 1-4 to represent four possible base hits:

  • 50% of hits are singles: return 1
  • 25% of hits are doubles: return 2
  • 10% of hits are triples: return 3
  • 15% of hits are home runs: return 4

Call your function a few times and print the names of the outcomes. (You can use a list for this, with the strings for out, single, double, triple, and home run. This is better than a long chain of if/elif/else statements.)
