A new part of the language, now: you can convert a letter to a number and a number to a letter, with the number being the position of the letter in the alphabet.

Notice that you get the same number for upper-case and lower-case letters; but when you convert to a letter you get the upper-case letter. And any character that isn't a letter has zero as a code, so you can distinguish letters from non-letters.

string letterA = "A"; number codeA = code for letterA; print codeA; string letterZ ="Z"; number codeZ = code for letterZ; print codeZ; string letterJ = letter for 10; print letterJ; string letterK = letter for 10 + 1; print letterK;
letterA = "A" codeA = code(letterA) print codeA letterZ ="Z" codeZ = code(letterZ) print codeZ letterJ = letter(10) print letterJ letterK = letter(10 + 1) print letterK
let letterA = "A"; let codeA = code(letterA); print(codeA); let letterZ ="Z"; let codeZ = code(letterZ); print(codeZ); let letterJ = letter(10); print(letterJ); let letterK = letter(10 + 1); print(letterK);
