In SCCL, you can loop through all the letters in a string with "for each letter variable in string". This statement defines a variable -- such as ltr in the code shown here -- and sets its value to each letter in the given string, one at a time, and for each of those values it runs the code that follows.

In Python, you can loop through all the letters of a string with for variable in string. This statement defines a variable -- such as ltr in the code shown here -- and sets its value to each letter in the given string, one at a time, and for each of those values it runs the code that follows.

In JavaScript, you can loop through all the letters of a string with for (let variable of string). This statement defines a variable -- such as ltr in the code shown here -- and sets its value to each letter in the given string, one at a time, and for each of those values it runs the code that follows.

After a loop statement, you must use curly braces { and } around the code you want to run for each letter. The code inside the braces is known as a "block" of code. You can put things all on one line, or on separate lines -- whatever seems right to you.

After a loop statement, you must include a colon :, and the "block" of code you want to run for each letter must go on the following line or lines. That line or all of those lines must be indented -- must start with some spaces or tabs -- in order to be controlled by the loop.

string message = "Hello"; for each letter ltr in message { print "Next letter is " + ltr; }
message = "Hello" for ltr in message: print "Next letter is", ltr
let message = "Hello"; for (let ltr of message) { print("Next letter is " + ltr); }
