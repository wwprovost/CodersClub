Variables can be strings of text, as well as numbers.

You can use the + operator to concatenate two strings: this means building a new string out of all the letters in the first string and then all the letters in the second string.

You can also use this when printing the results of a program, to print first a label and then a value, as shown below.

You can also use the functions int() to convert a string to a number and str() to convert a number to a string.
