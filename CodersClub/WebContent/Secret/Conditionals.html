Use an if statement to test that something equals something else, or does not equal something else. Notice the double-equals for testing equality -- not the same as the single-equals used for assigning a value to a variable. This must be followed by a block of code, just the way a loop is followed by a block -- but with an if, that block will be executed if the condition is true, and not executed otherwise.

An if can be followed by an else statement, and a second block of code, which will execute only if the first one does not.

string message = "Hello"; if message == "Hello" { print "Hello equals Hello!"; } if message != "Goodbye" { print "Hello is not Goodbye."; } // Try changing this value: number cents = 99; if cents == 100 { print "1.00"; } else { print "0." + cents; }

let message = "Hello"; if (message == "Hello") { print("Hello equals Hello!"); } if (message != "Goodbye") { print("Hello is not Goodbye."); } // Try changing this value: let cents = 99; if (cents == 100) { print("1.00"); } else { print("0." + cents); }
