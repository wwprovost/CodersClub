Learning the syntax of a new language can be tricky. Number Crunch has a "code assistant" that can help. When you hold down the Ctrl key and hit the Space key, a window will pop up with common phrases. Try this now: click in the program area, and then hit Ctrl-Space. Then double-click any of the statements in the list that pops up -- or use arrow keys to select it and hit Enter. That example is copied and pasted into your program code.

Often, some part of the statement will be highlighted -- so, the next thing you type will replace that highlighted part. You may still have to edit some other part of the example to get what you want.

Be sure you have the editing cursor (the spot in the code where you're typing) in the right place before starting the assistant, because that's where it will paste in the code.
