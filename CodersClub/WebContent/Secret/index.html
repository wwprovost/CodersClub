Secret Coders' Club

Welcome to the Secret Coders' Club. In these activities, you'll learn a simple programming language that was created just for the purpose of "encoding" and "decoding" messages. Once you've learned the language, you'll be able to write programs that take ordinary text, like this ...

WELCOME TO THE CLUB

... and turn it into what looks like gibberish ...

QYFWIGY NI NBY WFOV

... but is really an encoded version of the original message. And, with another program, you'll be able to decode the message again.

You'll also have the option of working with popular programming languages such as Python and JavaScript.
