Can you write a program that decodes the string below? It was created by adding one to the code of each letter, so you need to subtract one from the code of each letter in the string, and then build a new string.

This is a big exercise, and you'll need just about everything that you've learned so far: strings, numbers, arithmetic, loops, conditionals, code-for and letter-for conversions.

When you've got it, copy your whole program, except the first line, to the clipboard, so that you can paste it back in on the next page.
