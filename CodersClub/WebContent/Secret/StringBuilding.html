You can create new strings by adding letters to existing ones. This program creates an exact copy of one string, in another string. See how the copied string takes shape?

First, try to change the program so that you only print the completed copy, and not every version of it during the loop. And then: can you make the copied string "spread out," with a space inserted between each letter of the original string?

string message = "Hello"; string copy = ""; for each letter ltr in message { copy = copy + ltr; print copy; }
message = "Hello" copy = "" for ltr in message: copy = copy + ltr print copy
let message = "Hello"; let copy = ""; for (let ltr of message) { copy = copy + ltr; print(copy); }
