Here's a trick: you can get a number to move back and forth between two values, as shown below.

This can be useful in a loop, when you want to do something every other time, odds times and even times, etc.

number toggle = 1; print toggle; toggle = 3 - toggle; print toggle; toggle = 3 - toggle; print toggle; toggle = 3 - toggle; print toggle;
toggle = 1 print toggle toggle = 3 - toggle print toggle toggle = 3 - toggle print toggle toggle = 3 - toggle print toggle
let toggle = 1; print(toggle); toggle = 3 - toggle; print(toggle); toggle = 3 - toggle; print(toggle); toggle = 3 - toggle; print(toggle);
